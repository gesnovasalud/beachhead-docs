# Escritorio

## Descripción General
El Escritorio es la página principal de la aplicación del Hospital de Quilpué. Desde aquí puede acceder a todas las funcionalidades del sistema.

## Funcionalidades
- Visualización de resumen de actividades
- Acceso rápido a las principales secciones
- Notificaciones y alertas del sistema

## Navegación
Desde el escritorio puede acceder a:
- Solicitudes
- Agendas (Prequirúrgica y Quirúrgica)
- Encuentros (Prequirúrgico, Quirúrgico y Postquirúrgico)
- Monitores

## Búsqueda
Utilice la barra de búsqueda en la parte superior para encontrar rápidamente cualquier funcionalidad o información.
